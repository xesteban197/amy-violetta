# 🎀 Amy Violetta - Cumple 1 añito

Página web de invitación al primer cumpleaños de Amy Violetta.

## Archivos

```
amy-violetta/
├── index.html          # Estructura principal
├── style.css           # Estilos y animaciones
├── script.js           # Partículas flotantes
├── amy_violetta_bg.jpeg # Imagen de fondo
└── README.md
```

## Cómo abrir

1. Abre la carpeta en **Visual Studio Code**
2. Instala la extensión **Live Server** (si no la tienes)
3. Clic derecho sobre `index.html` → **Open with Live Server**

O simplemente haz doble clic en `index.html` para abrirlo directo en el navegador.
